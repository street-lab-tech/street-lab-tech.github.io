Bootstrap 5 migration files for streetlab.tech

1. Copy the files in this folder into the matching locations in your Jekyll repo.
2. Delete the old Bootstrap 3 assets and Sass source:
   - css/bootstrap.css
   - css/bootstrap.min.css
   - css/bootstrap-theme.css
   - css/bootstrap-theme.min.css
   - js/bootstrap.js
   - js/bootstrap.min.js
   - js/npm.js
   - fonts/glyphicons-halflings-regular.*
   - _sass/_bootstrap.scss
   - _sass/bootstrap/
3. Apply the same simple replacements already reflected in _pages/research/ if you prefer doing them by search/replace instead of copying those files one by one.
4. Rebuild the site locally and verify the layout at mobile, tablet, laptop, and desktop widths.
